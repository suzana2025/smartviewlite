# smartviewlite
Versão simples de monitoramento de produção com base na minha experiência, com documentação, análise de dados e testes
O **SmartView Lite** é um mini sistema inspirado no Projeto SmartView, onde simulo processos de produção industrial por meio do monitoramento diário de performance, produtividade e metas. 

Este projeto representa minha experiência prática na empresa Terrafós Nutrição Animal, Indústria, onde trabalhei com o supervisório Elipse, relatórios de produção, análise de indicadores e identificação de falhas.

Através dele represento conhecimento nas seguintes áreas:

- análise de dados
- documentação técnica
- testes de software
- suporte técnico
- entendimento de processos industriais
- Python básico

As funcionalidades presentes são:

- Leitura de dados de produção
- Cálculo de produtividade
- Dashboard gerado por Python
- Gráfico de performance
- Relatório final da produção
- Registro de possíveis inconsistências nos dados
